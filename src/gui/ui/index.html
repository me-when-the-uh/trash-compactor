Trash Compactor GUI

Where we're going, we don't need backups!

Report problems to: https://github.com/me-when-the-uh/trash-compactor.

An intelligent graphical interface for Windows NTFS filesystem compression using entropy-based heuristics.

For use with files and programs that rarely change - any file modifications will undo the compression for that file, so re-running this tool periodically is a good idea.

Vibe-coded in Python with a webview-slopped GUI.
